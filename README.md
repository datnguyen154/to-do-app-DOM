# 📝 To-Do App (DOM JavaScript)

[![Live Demo](https://img.shields.io/badge/demo-live-green)](https://datnguyen154.github.io/to-do-app-DOM/)


A simple and interactive **To-Do App** built using **HTML, CSS, and Vanilla JavaScript (DOM manipulation)**.
This project focuses on core JavaScript skills and improving user experience without using any frameworks.

---

## 📌 Features

* ➕ Add new tasks
* 🗑️ Delete tasks
* ✅ Mark tasks as completed
* ✏️ Edit tasks (inline editing)
* ⌨️ Press **Enter** to save while editing
* 🔄 Automatically update task count
* 🎯 Clean UI and smooth UX interactions
* 📦 Organized project structure (separate CSS & JS folders)

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS - DOM)

---

## 💡 Key Highlights

* Built without any frameworks (pure DOM manipulation)
* Implemented **inline editing experience**
* Added **keyboard support (Enter to save)**
* Handled dynamic UI updates efficiently
* Organized logic for better readability and maintainability

---

## 🧠 Architecture

* DOM-based rendering
* Event-driven interactions
* No frameworks (Vanilla JavaScript)
